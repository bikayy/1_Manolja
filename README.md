# Manolja
### : 숙박업소 예약 및 관리 프로그램
#### 로그인 권한에 따라 고객/관리자 메뉴가 별도 출력됩니다. 
#### 공통기능으로는 지역을 선택하고 숙박 일정을 입력하여 예약 가능한 숙소를 검색할 수 있습니다.
#### 고객은 숙소를 예약할 수 있고 예약내역 확인이 가능합니다.
#### 관리자는 업체를 등록한 후 업체별 객실을 등록할 수 있으며 업체별 고객 예약정보 또한 조회할 수 있습니다.
